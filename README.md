# Birthday Web Page (GitHub Pages)

This is a simple static birthday page (no database, no backend).

## Customize your message

Open `index.html` and replace:

- `[Her Name]`
- `[Your Name]`
- The birthday message text

You can also edit colors and style in `styles.css`.

## Run locally (optional)

You can just open `index.html` in a browser, or use VS Code Live Server.

## Publish on GitHub Pages

1. Create a new GitHub repository (for example: `birthday-page`).
2. Upload all files from this folder:
   - `index.html`
   - `styles.css`
   - `README.md`
3. In GitHub, go to **Settings → Pages**.
4. Under **Build and deployment**:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` and folder `/ (root)`
5. Click **Save**.
6. Wait 1–2 minutes, then refresh Pages settings to get your live URL.

Your site URL will look like:

- `https://<your-username>.github.io/<repo-name>/`

## If you want a cleaner URL

If your uni account gives a user site, create repo exactly named:

- `<your-username>.github.io`

Then the page URL becomes:

- `https://<your-username>.github.io/`
